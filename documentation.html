NOTEWORTHY

  My project uses a custom built canvas image managing system. The main application is designed with a general game engine framework in mind. In addition to the development of a robust object management system I have also used custom drawn images using GIMP. I have also used Bulma for the general framework of the project and web components for a globally consistent application.

GRADING

  My project goes above and beyond in the use of tools and skills learned in class, but where the highlight of this project is in my opinion is in the javascript. My project shows my experience in working with complex systems and my skill in javascript by managing well over 100 objects that are used in the game from fonts and buttons to sprites and enemy types. All of these elements are built into a single render list, scene manager, and game manager. From these managers the main application builds an experience of thrill as you fend off hordes of monsters one bullet at a time.

TO DO

  There are so many different directions that the project can take at this point. Currently it is a good proof of concept for an arena shooter built in javascript. With the foundation in place further expansion can be more enemy types with more variety. More weapon types with additional abilities such as bullet spread and piercing ammo. Kill streak multipliers. Better images and the addition of animations. Finally player customization such as player armor, UI, and arena colors, armor shape and weapon icons.
